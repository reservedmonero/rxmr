# rXMR Branding

## Runtime Identity

| Surface | Value |
|---|---|
| Daemon binary | `rxmrd` |
| Wallet CLI | `rxmr-wallet-cli` |
| Wallet RPC | `rxmr-wallet-rpc` |
| URI scheme | `rxmr:` |
| Default datadir | `~/.rxmr` |
| Chain name | `rxmr` |

## Currency Units

| Decimal point | Unit |
|---|---|
| 12 | `rxmr` |
| 9 | `millirxmr` |
| 6 | `microrxmr` |
| 3 | `nanorxmr` |
| 0 | `picorxmr` |

## Notes

- The chain was formerly launched as Bonero.
- The rename changes the public runtime surface, but the live mainnet preserves the existing genesis transaction and history.
