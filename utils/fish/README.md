## Fish shell completions for rXMR
This folder has basic Fish completions for `rxmrd`, `rxmr-wallet-cli`, and `rxmr-wallet-rpc`. To use them, put those files (or symlink them) inside `~/.config/fish/completions/` or wherever your Fish completion files are (see [https://fishshell.com/docs/current/completions.html#where-to-put-completions](https://fishshell.com/docs/current/completions.html#where-to-put-completions))
